# 实现复制数据功能

## 1. 添加设置项

### 修改 `types/setting.d.ts`
- 在 `Item` 接口中添加 `copyData` 字段，类型为 `boolean`

### 修改 `commons/utils/setting.ts`
- 在 `getItem` 函数中添加 `copyData` 参数，默认值为 `true`
- 在返回的 `Item` 对象中添加 `copyData` 字段

### 修改 `src/pages/setting/Index.vue`
- 在项目设置页面的功能部分添加 "复制数据" 复选框
- 绑定到 `setting.item.copyData`

## 2. 实现复制数据功能

### 修改 `electron/main/item/ipcEvent.ts`
- 添加 `copyData` 函数，实现以下逻辑：
  - 只对项目类型是文件生效
  - 获取项目文件所在目录
  - 在目录下创建 `.wstart` 目录
  - 复制项目图标到 `.wstart` 目录，命名为 `{项目名称}_ico.png`
  - 复制项目背景图到 `.wstart` 目录，命名为 `{项目名称}_cover.{扩展名}`
  - 创建 `wstart.ini` 文件，包含启动信息
- 在 `addItem` 和 `updateItem` 函数中，当 `setting.item.copyData` 为 true 时调用 `copyData` 函数
